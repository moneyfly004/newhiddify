# 赛博朋克 UI 设计指南

## 🎨 配色方案

### 主色调

- **霓虹青色** (`#00FFFF`) - 主要强调色
- **霓虹粉色** (`#FF00FF`) - 次要强调色
- **霓虹绿色** (`#00FF41`) - 成功/连接状态
- **霓虹黄色** (`#FFF700`) - 警告/排名第一
- **深色背景** (`#0A0A0A`) - 主背景
- **深色表面** (`#1A1A1A`) - 卡片背景
- **深色变体** (`#2A2A2A`) - 输入框背景

### 使用场景

- **青色**：主要操作、链接、边框
- **粉色**：次要操作、聚焦状态
- **绿色**：成功状态、连接状态
- **黄色**：警告、排名标识

## ✨ 视觉效果

### 发光效果

```dart
BoxShadow(
  color: neonCyan.withOpacity(0.5),
  blurRadius: 20,
  spreadRadius: 2,
)
```

### 渐变背景

```dart
LinearGradient(
  colors: [
    neonCyan.withOpacity(0.1),
    neonPink.withOpacity(0.1),
    neonGreen.withOpacity(0.1),
  ],
)
```

### 霓虹边框

```dart
Border.all(
  color: neonCyan.withOpacity(0.3),
  width: 1,
)
```

## 🎯 组件设计

### 按钮

- 使用渐变背景
- 添加发光阴影
- 粗体文字，增加字间距

### 卡片

- 深色背景
- 霓虹边框
- 可选渐变背景

### 输入框

- 深色背景
- 霓虹边框
- 聚焦时边框变粉色

### 连接按钮

- 大圆形设计
- 径向渐变
- 强烈发光效果

## 📐 布局原则

1. **间距**
   - 卡片间距：16px
   - 内容内边距：16px
   - 元素间距：8-12px

2. **圆角**
   - 卡片：16px
   - 按钮：12px
   - 输入框：12px

3. **字体**
   - 标题：粗体，增加字间距
   - 正文：正常粗细
   - 强调：使用霓虹色

## 🎭 动画效果

1. **模式切换**
   - 滑动动画
   - 300ms 缓动

2. **状态变化**
   - 颜色过渡
   - 发光效果变化

3. **加载状态**
   - 霓虹色加载指示器

## 💡 设计提示

1. **对比度**
   - 确保文字可读性
   - 使用白色文字在深色背景

2. **层次感**
   - 使用发光效果突出重要元素
   - 渐变增加深度感

3. **一致性**
   - 统一使用配色方案
   - 保持视觉风格一致
